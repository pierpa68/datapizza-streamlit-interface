WINDOWS 11
------------------------------------------------------------

  1. Scarica e decomprimi questo file ZIP
  2. Tasto destro su INSTALLA_DeepAiUG.bat
     -> "Esegui come amministratore"
  3. Segui le istruzioni a schermo

  Avvio: icona DeepAiUG sul Desktop
  Problemi: lancia CHECK_DeepAiUG.bat


MAC
------------------------------------------------------------

  1. Apri Terminale (Applicazioni -> Utility -> Terminale)
  2. Trascina installa_deepaiug_mac.sh nel Terminale e premi INVIO
     Oppure: bash installa_deepaiug_mac.sh
  3. Segui le istruzioni a schermo

  Prima apertura: tasto destro su DeepAiUG.command -> "Apri"
  Problemi: bash check_deepaiug.sh


LINUX (Ubuntu, Fedora, Arch)
------------------------------------------------------------

  1. Apri il Terminale
  2. chmod +x installa_deepaiug_linux.sh
  3. ./installa_deepaiug_linux.sh
  4. Segui le istruzioni a schermo

  Avvio: icona DeepAiUG nel menu applicazioni
         oppure: bash ~/DeepAiUG/avvia_deepaiug.sh
  Problemi: bash check_deepaiug.sh


PROBLEMI COMUNI
------------------------------------------------------------

  Cosa vedi?                 Cosa fare?
  -------------------------  ------------------------------------------
  Download lento             Normale: il modello AI pesa 2-5 GB
  Chat non risponde          Verifica che Ollama sia attivo
  Errori rossi               Lancia lo script di diagnostica per il tuo OS
  App non si avvia           Controlla la connessione Internet

  Log completo installazione in:
    Windows: C:\Users\[TUO NOME]\DeepAiUG_install_log.txt
    Mac/Linux: ~/DeepAiUG_install_log.txt
